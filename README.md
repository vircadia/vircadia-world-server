# Vircadia World Server

## Installation

### Supabase CLI

You need a relatively new version of `npm` so we can use the `npx` commands.

#### macOS

```bash
brew upgrade supabase
```

#### Linux

```bash
brew upgrade supabase
```

### Deno

#### macOS

```bash
brew install deno
```

#### Linux

```bash
sudo apt install deno
```

### Caddy

#### macOS

```bash
brew install caddy
```

#### Linux

```bash
sudo apt install caddy
```
