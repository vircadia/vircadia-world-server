# Vircadia World Server

## Installation

### Supabase CLI

#### macOS

```bash
brew upgrade supabase/tap/supabase
```

#### Linux

```bash
brew upgrade supabase/tap/supabase
```

### Deno

#### macOS

```bash
brew install deno
```

#### Linux

```bash
sudo apt install deno
```

### Caddy

#### macOS

```bash
brew install caddy
```

#### Linux

```bash
sudo apt install caddy
```
